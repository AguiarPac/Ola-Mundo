# Aprendendo a linguagem Markdown

risco: ~~risco~~
misturado: ~~***misturado***~~

 
 Lista numerada: (1.)
 1. lista 1
 2. lista 2
 3. lista 3
    1. lista 3.1
 
 Lista demarcada: (* ou -)
 - lista 1
 - lista 2
 - lista 3
   - lista 3.1
 
 Lista de tarefas:
 - [ ] tarefa 1
